# 🎮 Rock Paper Scissors Lizard Spock

A modern implementation of the classic **Rock Paper Scissors Lizard Spock** game, featuring both **single-player** and **online multiplayer** modes. Players can challenge an AI opponent with multiple difficulty levels or create online rooms and compete with friends in real-time using unique room codes.

The project is fully responsive, runs entirely in the browser, and provides an engaging user experience through animations, score tracking, leaderboards, and real-time game synchronization.

## ✨ Features

### 🎮 Create Game

* Play locally against an AI opponent
* Three AI difficulty levels:

  * Easy (Random - Pure luck)
  * Medium (Adaptive - Learns your patterns)
  * Hard (Expert - Advanced strategy)
* Host online multiplayer games
* Generate unique 6-digit room codes
* Customizable match lengths:

  * Single Game
  * Best of 3
  * Best of 5
  * Best of 7

### 🌐 Join Online Games

* Join games using room codes
* Real-time waiting room experience
* Automatic game synchronization between players
* Seamless multiplayer gameplay

## 🎯 Game Rules

The game follows the complete **Rock Paper Scissors Lizard Spock** ruleset:

| Choice      | Defeats          |
| ----------- | ---------------- |
| 🪨 Rock     | Scissors, Lizard |
| 📄 Paper    | Rock, Spock      |
| ✂️ Scissors | Paper, Lizard    |
| 🦎 Lizard   | Spock, Paper     |
| 🖖 Spock    | Scissors, Rock   |

### Winning Actions

* 🪨 Rock crushes Lizard
* 🪨 Rock crushes Scissors
* 📄 Paper covers Rock
* 📄 Paper disproves Spock
* ✂️ Scissors cuts Paper
* ✂️ Scissors decapitates Lizard
* 🦎 Lizard poisons Spock
* 🦎 Lizard eats Paper
* 🖖 Spock smashes Scissors
* 🖖 Spock vaporizes Rock

## 🚀 Deployment

This project is deployed using **GitHub Pages**.

### Steps Used

1. Push the project to GitHub.
2. Open the repository.
3. Navigate to:

   * **Settings** → **Pages**
4. Under **Build and Deployment**:

   * Source: **Deploy from a branch**
5. Select:

   * Branch: `main`
   * Folder: `/ (root)`
6. Save the changes.
7. GitHub automatically builds and publishes the website.

Our application becomes available at:

```text
https://aliben91.github.io/RockPaperScissorsLizardSpockGame/
```

Where aliben91 is the username and RockPaperScissorsLizardSpockGame is the repository-name
